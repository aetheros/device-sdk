Detailed Description

The SDK Application guard object.

An object of this type should be instantiated at the beginning of the toplevel scope at which AOS SDK is used. This would usually be near the beginning of the main() function. The constructor performs some global SDK initialization, and the destructor performs SDK cleanup.

Examples
aos_metering_app.cpp, and aos_neutral_app.cpp.
