Detailed Description

template<typename T>
class xsd::XsdFundamentalType< T >

XSD Fundamental type wrapper class.

This a a wrapper class for builtin schema types (like, xs:integer, xs:string) that are represented directly by a C++ type (std::int32_t, std::string).
