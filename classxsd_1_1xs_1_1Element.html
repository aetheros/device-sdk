Detailed Description

template<typename T, unsigned MinOccurs = 1, unsigned MaxOccurs = 1>
class xsd::xs::Element< T, MinOccurs, MaxOccurs >

The base class of elements (members) of XSD types where minOccurs and maxOccurs are not greater than 1.
