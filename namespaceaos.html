◆ getLocation()

aos::Json aos::getLocation ( RpcResult &  result = detail::rpcThrowOnError)

Retrieve the Device Location information.

Get the Device's location in json format. If no location data is available, json null is returned. If available, a json object with the following fields: lat: Latitude (floating point, degrees) lon: Longitude (floating point, degrees) alt: Altitude (floating point, meters) rad: Horizontal Uncertainty (floating point, meters) ts: Timestamp (Unix Epoch seconds)

Returns
A Json object with location data, or null.
