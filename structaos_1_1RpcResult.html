Detailed Description

A class containing results of an rpc request.

Api calls which perform RPC have an optional RpcResult parameter. When the RpcResult parameter of these functions is provided with a reference to an RpcResult object, then in the case of an rpc error, the RpcResult object instance will be populated with error information and the function WILL NOT throw an exception. If no reference to an RpcResult is provided, the function will throw an exception due to an RPC error. An exception may still occur in the event of a non-rpc-related error.
